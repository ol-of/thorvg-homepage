Thor Vector Graphics

A lightweight engine for vector graphics and animation.

ThorVG is an open-source library for rendering vector scenes and animations including SVG and Lottie. The renewed site keeps the existing content, but makes it easier to move between overview, documentation, tools, and examples.

Core strengths

Small, practical, and broad enough for real use.

ThorVG focuses on rendering capabilities that matter in products: compact integration, multiple backends, support for common vector formats, and efficient updates for dynamic scenes.

Structural Design

One engine, modular composition

A building-block design keeps binary size and maintenance pressure under control.

Threading

Parallel work where it matters

Task scheduling is built to help update, decode, and render scenes efficiently.

Smart Rendering

Less redraw, more useful work

Partial updates make dynamic graphics and UI rendering significantly more efficient.

Formats

SVG and Lottie first-class

Static vector scenes and modern animation workflows live in one engine.
